Ordem para criar um repositorio:
- Inicializar o repositorio:
```bash
git init
```

- Adicionar os arquivos:
```bash
git add.
```

- Commitar (salvar) os arquivos:
```bash
git commit -m "mensagem"
```
- Verificar status:
```bash
git status
```
